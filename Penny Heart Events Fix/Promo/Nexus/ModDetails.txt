Details:
[b]Description:

[color=#ffff00]Penny Heart Events Fix[/color][/b] is a Content Patcher mod which allows Penny's 4- and 6-heart events to trigger in Pam's house after the Community Upgrade. The only changes from the events in the trailer are adjusting character movement to fit the new location and removing the requirement that Penny already be home. Supports all standard game languages (English, German, Spanish, Portugese, Russian, Japanese and Chinese.)


[b]Installation:[/b]
[list=1]
[*]Install the latest version of [url=https://smapi.io/]SMAPI[/url].
[*]Install the latest version of [url=https://www.nexusmods.com/stardewvalley/mods/1915]Content Patcher[/url].
[*]Unzip the download and place the included [color=#b6d7a8][code][CP] Penny Heart Events Fix[/code][/color] folder into your [color=#b6d7a8][code]Mods[/code][/color] folder.
[/list]


Compatibility:
[/b]
[b][color=#ffff00]Penny Heart Events Fix[/color][/b] requires [url=https://www.nexusmods.com/stardewvalley/mods/1915]Content Patcher[/url] version 1.4 or higher (which means it also requires [url=https://smapi.io]SMAPI[/url] version 2.6 or higher). It will only work on Stardew Valley version 1.3 since the Community Upgrade was added by that version.
